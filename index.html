API JSON Reader

API JSON Reader (або APIJR) — утиліта для роботи з API Єдиного веб-порталу використання публічних коштів [spending.gov.ua], що дозволяє швидко знаходити й обробляти транзакції, договори, бюджетні зобов'язання та інші сервіси, конвертувати дані у таблиці, CSV або SQL, створювати сценарії запитів і зберігати результати у різних форматах.

Також, APIJR надає користувачеві доступ до ресурсів Порталу відкритих даних [data.gov.ua], пропонуючи зручні інструменти для ефективної роботи з різноманітними наборами даних.

Ось кілька ключових особливостей:

Наш додаток значно спрощує, пришвидшує та автоматизує пошук даних за транзакціями, розпорядниками чи зобов’язаннями — без необхідності відвідувати Єдиний веб-портал використання публічних коштів, який надає доступ до своїх даних через публічне АРІ (ліцензія Creative Commons Attribution 4.0 International). Також, APIJR робить більш комфортною роботу з наборами даних, які надає Портал відкритих даних (ліцензія Creative Commons Attribution 4.0 International). Кількість доступних користувачеві наборів даних може змінюватися з часом і доступна через головне меню застосунку в пункті «Портал відкритих даних».

Інтуїтивний інтерфейс та простота використання

APIJR створений без зайвого функціоналу, з акцентом на зручність. Навіть новачки легко розберуться в структурі запитів і роботі з довідниками порталу Spending чи наборами Порталу відкритих даних.

Автоматична обробка та збереження даних

Застосунок дозволяє не лише переглядати результати запитів, а й автоматично конвертувати отримані JSON у таблиці, CSV або SQL, зберігаючи файли в потрібному кодуванні (для API Spending). Набори даних порталу data.gov.ua також доступні у вигляді зручної таблиці, бази даних SQLite та CSV файлів

Гнучкість і сценарії для роботи з API

Працюючи з API Spending, користувачі можуть налаштувати параметри запитів, запускати сценарії одним кліком і отримувати дані вже в зручному для аналізу вигляді — без потреби у повторному введені параметрів часто використовуваного запиту до API. Раніше завантажені набори даних data.gov.ua зберігаються в локальній базі даних, тому завжди доступні для роботи навіть за відсутності інтернету

З API JSON Reader (APIJR) ви отримуєте зручний інструмент для швидкої та ефективної роботи з відкритими фінансовими даними Порталу використання публічних коштів та Порталом відкритих даних, що дозволяє аналізувати, фільтрувати та експортувати інформацію в кілька кліків. Завдяки автоматизації рутинних дій (конвертація, збереження, SQL-генерація), ви економите час і отримуєте максимум користі з публічних фінансів та відкритих даних — без потреби в складному програмуванні чи ручній обробці CSV, XML чи JSON-файлів.

Ось як виглядає інтерфейс користувача

APIJR не перевантажений зайвим функціоналом, який міг би заплутати користувача. Інтерфейс програми інтуїтивно зрозумілий навіть для новачків. Як і на порталі використання публічних коштів, ви можете переглядати всі доступні довідники системи та формувати запити для пошуку даних на основі вхідних параметрів. Крім цього, ви можете налаштувати власний механізм постобробки отриманих даних: вибрати кодування, автоматично розбивати результати по контрагентах, задавати параметри збереження у файл, а також генерувати таблицю, CSV-файл або SQL-запити для вставки в базу даних. Повний функціонал застосунку доступний за підпискою APIJR Pro.

Налаштування обробки даних

Ви самі обираєте, які механізми постобробки застосовувати до отриманих даних.

Табличне представлення даних

Таблиця створюється автоматично з отриманого JSON. Її можна скопіювати у будь-який табличний редактор або зберегти як файл Excel.

Формування CSV з JSON файлу

Формування CSV (як і SQL) з отриманого JSON файлу також відбувається автоматично, якщо ви обрали таку опцію в головному меню програми.

Перегляд набору даних з Порталу відкритих даних

Інформація з державного реєстру судових рішень за 2007 рік у вигляді зручної таблиці, яку можна фільтрувати, виконувати пошук, сортувати, копіювати часткову або повну інформацію до буфера обміну, експортувати дані в Excel (автоматично сформується необхідна кількість аркушів у книзі Excel)

Використовуючи наш застосунок, ви зберігаєте повний контроль над своїми даними. Ми не збираємо жодної особистої інформації та не передаємо її на наш сервер чи сторонні ресурси. Уся робота програми відбувається локально — лише на вашому комп’ютері — у прямій взаємодії з офіційним API порталу використання публічних коштів.

Відкрити в Microsoft Store

Які дані доступні через портал Spending

Spending.gov.ua — офіційний державний вебпортал відкритих даних про використання публічних коштів розпорядниками та одержувачами бюджетних коштів державного і місцевих бюджетів. Він є складовою частиною платформи відкритих даних у сфері публічних фінансів E-data. Портал надає громадськості доступ до інформації про витрачання коштів: розпорядниками та одержувачами бюджетних коштів; державними та комунальними підприємствами; Пенсійним фондом України; фондами загальнообов’язкового державного соціального страхування.


На порталі у табличній та графічній формах представлено:


📊 Фінансові дані: бюджетні зобов’язання; платіжні трансакції на єдиному казначейському рахунку.

📄 Договори: укладені договори та додаткові угоди; акти/накладні щодо виконання договорів; сплата штрафних санкцій за порушення умов договорів.

📈 Звітність: обсяги платежів за договорами в розрізі товарів, робіт і послуг; витрати на службові відрядження; виконання бюджету Пенсійного фонду та фондів соцстраху; форми бюджетної звітності (№2, 4-1, 4-2, 4-3, 4-3.1, 4-4, 7, 7.1); звіт про підприємства, що використовують кошти фондів.

🧾 Зведена аналітика: кількість та структура оприлюднених документів; кількість і сума трансакцій; розподіл розпорядників та одержувачів за областями та КОПФГ; активність відвідувачів порталу.

Які дані доступні через Портал відкритих даних

Data.gov.ua — портал відкритих даних, створений з метою забезпечення вільного доступу до публічної інформації у формі відкритих даних. Він є частиною національної інфраструктури відкритих даних України та адмініструється Міністерством цифрової трансформації. Портал надає централізований доступ до наборів відкритих даних, опублікованих органами виконавчої влади, місцевого самоврядування, державними підприємствами, установами та організаціями. Серед іншого, користувачі можуть знайти інформацію про реєстри, ліцензії, судові рішення, екологічні дані, транспорт, фінанси, освіту, медицину та інші сфери суспільного життя.


На порталі представлені різноманітні набори даних за такими категоріями:


🏛️ Держава: все що стосується державного управління та питань загальнодержавного характеру.

💰 Державні доходи і видатки: закупівлі, витрати, бюджетні показники.

📈 Економіка та бізнес: все про компанії, питання пов'язані з веденням господарської діяльності.

⚡ Енергетика: тарифи, видобування, споживання, імпорт/експорт енергоносіїв тощо.

🚉 Інфраструктура і транспорт: транспортні засоби, маршрути, перевезення, ремонт та будівництво шляхів.

🌳 Навколишнє середовище: екологія, тваринний/рослинний світ, свердловини, водні ресурси, якість повітря/води, клімат тощо.

🎓 Освіта, культура, спорт: навчальні заклади, освітні програми, культурні/історичні пам'ятки, інфо-матеріали (виробництво/розповсюдження), кіно-, аудіо-індустрія, активний відпочинок, спортивна діяльність.

🩺 Охорона здоров'я: захворювання, лікарські засоби, лікарні, формуляри тощо.

🌍 Регіональний розвиток: територіальні громади, комунальна власність, місцевий розвиток.

👥 Суспільство: громадянське суспільство, соціальні стандарти та захист, пільги тощо.

💳 Фінанси: державні гарантії, звіти, бюджетні програми тощо.

⚖️ Юстиція та судочинство: нормативно-правові засади діяльності держави, суди та судочинство, правоохоронна діяльність.
